# NayePankh Foundation Landing Page

A modern and responsive landing page created for the NayePankh Foundation Front End Development Internship Assignment.

## Overview

This project is designed to showcase a clean, user-friendly, and responsive website for NayePankh Foundation. The webpage highlights the organization's mission, impact, internship opportunities, and contact information while providing an engaging user experience.

## Features

* Responsive Design for Desktop, Tablet, and Mobile Devices
* Modern Hero Section
* About NayePankh Foundation Section
* Impact Statistics Section
* Mission & Vision Section
* Internship Opportunities Section
* Contact Form
* Dark Mode Toggle
* Smooth Scrolling Navigation
* Interactive Hover Effects
* Scroll-Based Animations

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```text
nayepankh-foundation-landing-page
│

```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.

## Purpose

This project was developed as part of the Front End Development Internship selection process for NayePankh Foundation. The goal was to create a visually appealing and accessible webpage that effectively represents the organization's mission and activities.
